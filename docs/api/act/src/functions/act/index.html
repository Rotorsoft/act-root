Function: act()

act<TSchemaReg, TEvents, TActions, TStateMap, TActor>(): ActBuilder<TSchemaReg, TEvents, TActions, TStateMap, TActor>

Defined in: libs/act/src/builders/act-builder.ts:337

Creates a new Act orchestrator builder for composing event-sourced applications.

Examples​

const app = act()
  .withState(Counter)
  .build();
type MyActor = { id: string; name: string; role: string };

const app = act()
  .withActor<MyActor>()
  .withState(Counter)
  .build();
const CounterSlice = slice()
  .withState(Counter)
  .on("Incremented")
    .do(async (event) => { console.log("incremented!"); })
    .to("counter-target")
  .build();

const app = act()
  .withSlice(CounterSlice)
  .build();
